# Advanced AI Engineering for IT & Developer Teams

This repository contains the course materials, hands-on laboratories, and workshop exercises for the **Advanced AI Engineering for IT & Developer Teams** training program. The curriculum is organized into two learning modules that progressively cover machine learning, deep learning, prompt engineering, and modern generative AI application development.

#### Workshop Goals

Participants will:

- Use Gemini as a **coding assistant for data analysis**
- Build a full ML pipeline with AI support
- Learn how to debug AI-generated code
- Compare multiple models (LogReg, SVM, KNN, Random Forest)
- Identify data leakage and evaluation pitfalls

#### Suggested Flow

1. Start with the **Template notebook**
2. Use Gemini to complete each section:
    - Pandas exploration
    - Preprocessing
    - Model training
    - Evaluation
3. Compare with the **Solution notebook**
4. Discuss differences and improvements

---

## Learning Objectives

Upon completion of this course, participants will be able to:

* Perform data manipulation and preprocessing using Python and Pandas.
* Build and evaluate traditional machine learning models using Scikit-learn.
* Develop deep learning applications with PyTorch Lightning.
* Apply prompt engineering techniques for Large Language Models (LLMs).
* Build LLM-powered applications using LangChain and LangGraph.
* Track and evaluate LLM workflows using LangSmith.
* Utilize OpenAI Codex to accelerate software development.
* Develop Retrieval-Augmented Generation (RAG) applications and AI agents.
