# Advanced AI Engineering for IT & Developer Teams

This repository contains the course materials, hands-on laboratories, and workshop exercises for the **Advanced AI Engineering for IT & Developer Teams** training program. The curriculum is organized into two learning modules that progressively cover machine learning, deep learning, prompt engineering, and modern generative AI application development.

---

# Course Outline

## Day 1 — Machine Learning & Deep Learning

The first day focuses on the fundamentals of data science and artificial intelligence, beginning with data manipulation and preprocessing before progressing to machine learning, deep learning, and prompt engineering.

### Modules

* **1.1** Python & Pandas
* **1.2** Data Preparation
* **1.3** Traditional Machine Learning
* **1.4** Deep Learning
* **1.5** Prompt Engineering
* **1.6** IRIS Day 1 Workshop

---

## Day 2 — Generative AI

The second day introduces modern Generative AI development, including LLM frameworks, AI-assisted software engineering, and production-ready AI workflows.

### Modules

* **2.1** LLM Frameworks: LangChain, LangGraph, and LangSmith
* **2.2** OpenAI Codex (Standalone)
* **2.3** OpenAI Codex for Visual Studio Code (Full-Stack Software Development)
* **2.4** OpenClaw
* **2.5** IRIS Day 2 Workshop

---

## Learning Objectives

Upon completion of this course, participants will be able to:

* Perform data manipulation and preprocessing using Python and Pandas.
* Build and evaluate traditional machine learning models using Scikit-learn.
* Develop deep learning applications with PyTorch Lightning.
* Apply prompt engineering techniques for Large Language Models (LLMs).
* Build LLM-powered applications using LangChain and LangGraph.
* Track and evaluate LLM workflows using LangSmith.
* Utilize OpenAI Codex to accelerate software development.
* Develop Retrieval-Augmented Generation (RAG) applications and AI agents.

---

## Repository Structure

* **Day 1** – Machine Learning, Deep Learning, and Prompt Engineering materials
* **Day 2** – Generative AI frameworks, AI-assisted software development, and workshop exercises

Each module contains Jupyter notebooks, sample datasets, and practical exercises designed to provide hands-on experience with modern AI development tools and techniques.
